# Польовий тренажер алгоритмів

Демонстраційний сайт для заняття з Теорії алгоритмів (сортування, пошук, BFS/DFS) із візуалізацією кроків та підрахунком операцій.

## Як запустити
- Відкрийте `index.html` у браузері (мережевих залежностей немає).
- Керування: змініть розмір даних слайдером, генеруйте нові масиви, запускайте сортування/пошук/пошук шляху; кнопка **Скинути** повертає базові налаштування.

## Що показує
- Selection vs Quick sort: анімація барів, кількість операцій, час, кадри та теоретична складність.
- Лінійний vs двійковий пошук: кроки, індекси, знайдені/пропущені елементи, порівняння O(n) та O(log n).
- BFS vs DFS на сітці 8×8: генерація перешкод, відвідані клітинки та підсвічений шлях, статистика кроків/часу.
- Шпаргалка складності для швидкого нагадування.

## Структура
- `index.html` — каркас сторінки.
- `styles.css` — тема (темний фон, сітка, адаптивні панелі).
- `app.js` — логіка генерації даних, сортування, пошуку, BFS/DFS та оновлення UI.
